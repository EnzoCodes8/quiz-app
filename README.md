**ECE Board Exam Quiz Web Application**

A full-stack web-based quiz application designed for Electronics Engineering (ECE) board exam review. The platform provides randomized quizzes, score tracking, subject selection, and instant answer feedback for multiple ECE subjects.

Access Link: https://quiz-3g969s71e-enzocodes8s-projects.vercel.app/

Features
- Subject Selection
- Randomized Questions
- Instant Answer Feedback
- Score Recording
- Quiz History
- Responsive UI
- Online Deployment Support

Subjects Covered
- Mathematics
- General Engineering and Applied Sciences (GEAS)
- Electronics Engineering
- Electronics Systems and Technologies (EST)


**Tech Stack**
**Frontend**
- React
- CSS

**Backend**
- FastAPI
- Python

**Database**
- SQLite (Development)
- Supabase PostgreSQL (Production)

**Deployment**
- Vercel (Frontend)
- Render (Backend)
